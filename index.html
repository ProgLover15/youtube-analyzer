プライバシーポリシーおよび利用規約

本アプリケーション（以下「当サービス」）の利用にあたり、以下の事項をご確認ください：

  • YouTube API Servicesの利用: 当サービスはYouTube API Servicesを利用して登録チャンネル情報を取得しています。利用者は当サービスを利用することで、YouTube利用規約に拘束されることに同意したものとみなされます。
  • データの収集と保存: 当サービスは、利便性向上のためにお客様のブラウザのローカルストレージ（localStorage）を使用して、登録チャンネルのキャッシュデータおよびUIの設定を保存します。
  • データの更新と削除: 取得したAPIデータは最新の状態を保つため6時間ごとに自動的に更新（再取得）されます。また、ログアウト時やブラウザのキャッシュをクリアすることで、これらのデータはデバイスから削除されます。
  • Googleの規約: Googleによるデータの取り扱いについては、Googleプライバシーポリシーをご確認ください。
  • アクセスの取り消し: 当サービスへのアクセス許可は、いつでもGoogle セキュリティ設定ページから取り消すことができます。

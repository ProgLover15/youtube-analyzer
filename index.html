プライバシーポリシー

SubCleanerは、ユーザーのプライバシーを尊重し、YouTube API サービスから取得したデータの保護に努めます。当アプリを利用することで、ユーザーは Google プライバシーポリシー に同意したものとみなされます。

1. アクセスする情報の種類と目的

当アプリは、YouTubeの「登録チャンネル一覧」および「動画の更新情報」にアクセスします。これは休止中のチャンネルを特定して表示するためであり、他の目的には使用しません。

2. Cookie（クッキー）の使用について

当アプリは、ログイン状態の維持（セッション管理）のために、ユーザーのブラウザにCookieを配置、またはアクセスします。これにより認証情報の保護と利便性の向上を図ります。

3. データの保存と第三者提供

取得したAPIデータは表示の際に一時的にメモリ上で処理されますが、開発者のサーバーに保存したり、第三者に共有・販売したりすることはありません。

4. アクセス権の取り消しとデータ削除

ユーザーはログアウトすることでブラウザ上のセッション情報を削除できます。また、Google セキュリティ設定ページ から、いつでも本アプリのアクセス権を個別に直接取り消すことができます。

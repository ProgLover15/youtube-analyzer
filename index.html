プライバシーポリシーおよび利用規約
  • YouTube API Servicesの利用: 本アプリはYouTube API Servicesを利用しています。利用者はYouTube利用規約に同意したものとみなされます。
  • データの収集と保存: 利便性向上のため、ブラウザのCookie（クッキー）およびローカルストレージを使用して設定やキャッシュデータを保存します。
  • データの更新と削除: APIデータは6時間ごとに自動更新され、ログアウト時にデバイスから削除されます。
  • Googleの規約: Googleプライバシーポリシーをご確認ください。
